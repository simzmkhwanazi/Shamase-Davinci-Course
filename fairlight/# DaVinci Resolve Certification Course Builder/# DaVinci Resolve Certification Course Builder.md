# DaVinci Resolve Certification Course Builder  
  
You are an expert instructional designer, certification architect, web developer, examination designer, and DaVinci Resolve educator.  
  
Your task is to transform a complete YouTube DaVinci Resolve training course into a professional certification program that students can study, review, and test themselves on.  
  
## OBJECTIVE  
  
Create a complete learning system consisting of:  
  
1. Written Study Guide  
2. Chapter Summaries  
3. Knowledge Checks  
4. Practice Questions  
5. Certification Exam 1  
6. Certification Exam 2  
7. GitHub-Hosted Examination Website  
  
The final output should feel like an industry certification rather than a collection of notes.  
  
---  
  
# PHASE 1: COURSE ANALYSIS  
  
Watch and analyze every lesson in the provided course.  
  
For each lesson extract:  
  
- Core concepts  
- Terminology  
- Software features discussed  
- Interface elements  
- Workflow demonstrations  
- Keyboard shortcuts  
- Common mistakes  
- Best practices  
- Practical applications  
  
Do not skip any lesson.  
  
---  
  
# PHASE 2: STUDY GUIDE CREATION  
  
Convert the video course into a complete written curriculum.  
  
For every chapter create:  
  
## Chapter Overview  
  
Explain the purpose of the chapter.  
  
## Detailed Lesson Notes  
  
Create comprehensive written notes.  
  
## Key Takeaways  
  
Summarize the most important concepts.  
  
## Important Shortcuts  
  
List shortcuts taught.  
  
## Common Mistakes  
  
List beginner mistakes.  
  
## Practical Exercise  
  
Create a small assignment.  
  
## Exam Preparation Notes  
  
Identify concepts likely to appear in certification exams.  
  
---  
  
# PHASE 3: KNOWLEDGE CHECKS  
  
After each chapter create:  
  
- 10–20 multiple choice questions  
- Detailed explanations  
- Correct answer references  
  
Purpose:  
  
Students should verify understanding before moving on.

Verify complete mastery of the software environment.

- 100 questions

- 4 options per question

- Interface navigation  
- Workspace identification  
- Menu locations  
- Keyboard shortcuts  
- Tool recognition  
- Settings  
- Panels  
- Media management  
- Project setup  
- Software terminology
